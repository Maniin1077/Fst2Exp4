# EXP 2 - React Web Application

## Aim
To create a simple web application using React with Vite, Bootstrap, and Material UI.

## Description
In this experiment, a React project was created using Vite as the build tool.  
The application uses React Root for rendering and is structured using reusable components.  
Bootstrap is used for layout and responsiveness, while Material UI is used for modern UI components.

## Components Used
- Navbar
- Hero Section
- Cards
- Features Section
- Statistics Section
- Footer

## Tools & Technologies
- Vite
- React
- Bootstrap
- Material UI
- VS Code

## Output
A responsive and professional-looking single-page web application.

## Conclusion
This experiment helped in understanding how to build a modern web application using React and how to integrate UI libraries for better design.
